

## 2. 마스터 코드 테이블 (Master Data)

### A. 평직 계열 (Plain Weave / Group: P)

가장 기본적이며 튼튼한 1:1 교차 조직

| Code | 한글 명칭 | 영문 명칭 | 비고 (특징) |
| --- | --- | --- | --- |
| **PL** | 기본 평직 | Plain / Taffeta | 가장 기본적인 조직 (태피터, 보일 등) |
| **PO** | 옥스포드 | Oxford | 바스켓 변형, 통기성이 우수함 |
| **PB** | 바스켓 | Basket / Matt | 두 올 이상씩 교차, 바구니 짜임 |
| **PR** | 립스탑 | Ripstop | 격자 무늬, 찢어짐 방지 강화 |
| **PP** | 포플린 | Poplin | 위사가 경사보다 굵어 가로 골이 보임 |
| **PC** | 캔버스 | Canvas | 굵은 실로 촘촘하고 두껍게 짠 직물 |

### B. 능직 계열 (Twill Weave / Group: T)

대각선(사선) 무늬가 특징, 유연하고 구김이 적음

| Code | 한글 명칭 | 영문 명칭 | 비고 (특징) |
| --- | --- | --- | --- |
| **TW** | 기본 능직 | Basic Twill | 일반적인 사선 조직 (치노, 개버딘) |
| **TD** | 데님 | Denim | 경사(색사)와 위사(백사) 교차 |
| **TH** | 헤링본 | Herringbone | 청어 뼈 모양의 V자 사선 반복 |
| **TB** | 브로큰 트윌 | Broken Twill | 사선이 이어지지 않고 끊어지는 형태 |
| **TS** | 서지 | Serge | 45도 각도의 뚜렷한 사선 (소모 직물) |

### C. 주자직 계열 (Satin Weave / Group: S)

교차점을 최소화하여 매끄럽고 광택이 우수함

| Code | 한글 명칭 | 영문 명칭 | 비고 (특징) |
| --- | --- | --- | --- |
| **SA** | 새틴 | Satin | 견/합섬 사용, 광택 매우 우수 |
| **SN** | 면 주자 | Sateen | 면(Cotton) 방적사를 이용한 주자직 |
| **SC** | 크레이프 백 | Crepe Back Satin | 이면(Back)에 강연사(Crepe) 사용 |

### D. 변형 및 특수 조직 (Fancy & Special)

도비, 자카드 장치 사용 또는 파일(Pile) 직물

| Code | 한글 명칭 | 영문 명칭 | 비고 (특징) |
| --- | --- | --- | --- |
| **DO** | 도비 | Dobby | 소형 기하학적 무늬 반복 |
| **DP** | 피케 (PK) | Pique | 벌집 모양 등 올록볼록한 표면 |
| **JA** | 자카드 | Jacquard | 복잡하고 큰 무늬 제직 가능 |
| **VL** | 벨벳/벨로아 | Velvet / Pile | 표면에 털(Pile)이 있는 조직 |
| **CO** | 코듀로이 | Corduroy | 세로 방향의 골이 있는 파일 직물 |
| **SE** | 시어서커 | Seersucker | 장력 차이로 인한 주름(요철) 효과 |

### E. 니트/편물 (Knits / Group: K)

직물 시스템에서 니트 소재를 함께 관리할 경우 사용

| Code | 한글 명칭 | 영문 명칭 | 비고 (특징) |
| --- | --- | --- | --- |
| **KS** | 싱글 저지 | Single Jersey | 기본 티셔츠 원단, 단면 |
| **KI** | 인터록 | Interlock | 양면 조직, 가장자리 말림 없음 |
| **KR** | 립 (시보리) | Rib | 신축성이 뛰어난 고무단 조직 |
